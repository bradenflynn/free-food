## Verification Criteria
- Logging in as admin makes approval buttons appear.
- Clicking 'REJECT' instantly hides the event for the admin (and others once they refresh or via Realtime).

## must_haves
- Plain users (not logged in) never see the admin buttons.
- Session persists on refresh (Supabase handles this automatically if initialized correctly).
